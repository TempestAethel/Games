# Unwinnable-Tic-Tac-Toe
A tic tac toe where you can only lose or draw 

Game Modes:
- 2 Player Mode
- Random AI Mode
- Smart AI Mode (Minimax algorithm)
  
With a Reset Button 

All the file during each step is kept in the Prototypes folder
